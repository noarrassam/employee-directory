# Employee Directory

## Description

It is a React JS application that allows a user to view the entire employee directory at once so that a user can have quick access to their information.

## Used

- HTML

- CSS

- Bootstrap-CDN

- React JS

- Components

- React Hooks: useState, useEffect

- Localstorage

- API Fetch method
